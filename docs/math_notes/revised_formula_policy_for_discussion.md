## 1. 2.3節：指数型分布族の式をスカラーとして書くか

### 今回の予稿：スカラー表記を維持し，補足文を1行追加する

**現在の原稿の式：**
$$p(x \mid \eta) = h(x)\exp\{\eta T(x) - A(\eta)\}$$
この式のまま維持してよい。

**追加すべき補足文（2.3節末尾に1文）：**
> 「本稿では各観測値を1変量の指数型分布族として扱う．属性情報 $x_{il}$ については各属性次元 $l$ ごとに独立に適用する．」

**根拠：** y_ij はスカラー，x_il は次元ごとのスカラー → 1変量式で正確に表現できる。先行研究[1]のガウスも対角共分散なので次元独立の特殊ケース（Q1参照）。

### 修論・将来版での整理

多変量指数型分布族との関係（例: 共分散なし仮定の明示，成分独立性の仮定）を理論的に位置づける章を追加してもよい。ただし2ページ予稿では不要。

## 2. 3.1節：X側を x_il ごとの式で書くか，x_i ベクトルの式で書くか

### 今回の予稿：成分表示（x_il）で書き，従来との関係を1文補足する

**推奨する式（提案モデル生成モデル）：**
$$x_{il} \mid \mathbf{z}_i \sim \mathrm{ExpFam}_X(\eta_{il}^X), \quad \eta_{il}^X = \mathbf{f}_l^\top \mathbf{z}_i$$

**従来との接続を1文追加：**
> 「先行研究[1]が仮定するガウス分布（対角共分散 Σ）も，$x_{il} \sim \mathcal{N}(\mathbf{f}_l^\top \mathbf{z}_i, \sigma_l^2)$ と各次元独立なガウス分布として解釈できる特殊ケースである．」

**根拠：** 実装は `eta_x_i = F @ z_i`（(d,) ベクトル）に対して element-wise にリンク関数を適用しており，x_il の成分表示が最も正確。ベクトル記法にすると「全次元同一分布族」のように読める可能性があるため，成分表示の方が先生の疑問（"個別に指定"）に答えやすい。

**バイアス項 w_{0l} についても修正が必要：**  
現原稿の `η_{il}^X = w_{0l} + z_i^T w_l` は誤り。実装は `eta_x_i = F @ z_i`（バイアスなし）。  
→ `η_{il}^X = f_l^T z_i` に修正し，「先行研究[1]と同様，属性データのバイアスは0と仮定する」を一文追加。

### 修論・将来版での整理

x_i ベクトル全体に対する多変量指数型分布族への拡張（例: 混合分布族）や，成分間相関の導入を検討してもよい。今回の予稿では「各次元独立」という仮定を明示することで十分。

## 3. 3.2節：精度行列のY項に 1/2 を入れるかどうか

### 今回の予稿：1/2 を維持し，先行研究[1]との一致を明示する

**維持すべき式：**
$$\mathbf{A}_i = \mathbf{I}_k + \mathbf{F}^\top \mathbf{V}_X(\mathbf{m}_i)\,\mathbf{F} + \frac{(w^Y)^2}{2}\sum_{j \neq i} A_Y''(\eta_{ij}^Y)\,\mathbf{z}_j \mathbf{z}_j^\top$$

**追加すべき注記（本文か脚注に1文）：**
> 「第3項の 1/2 は先行研究[1]（式(22)）と同一の慣行に従う（Y 側の対数尤度を $(1/2)\sum_{i\neq j}$ として評価）．」

**根拠：**  
- `model_expfam.py` L.267: `ll += 0.5 * np.sum(ln_p)` → Y対数尤度は (1/2)Σ_{i≠j}
- `reproduction/src/model.py` L.306-307: "Based on paper Eq. (22): A_i = ... + (1/2)Σ_{j≠i}..."
- M-step 勾配: `grad_sum / (2.0 * L)` も同じ慣行
- 先生に「j≠i なら 1/2 不要では？」と言われたとき，「(1/2)Σ_{i≠j} で定義しているため必要」と答えられる

**もし先生が 1/2 除去を強く望む場合：**  
記法変更案（数学的に等価）：
$$\mathbf{A}_i = \mathbf{I}_k + \mathbf{F}^\top \mathbf{V}_X(\mathbf{m}_i)\,\mathbf{F} + (w^Y)^2\sum_{j > i} A_Y''(\eta_{ij}^Y)\,\mathbf{z}_j \mathbf{z}_j^\top$$
ただし M-step 勾配式も `Σ_{i<j} / L` に変更が必要（提出直前での変更はリスクあり）。

### 修論・将来版での整理

(1/2)Σ_{i≠j} と Σ_{i<j} の等価性を付録で証明するか，どちらかに統一した記法ガイドラインを定めるとよい。

## 4. θ に Σ を含めるかどうか

### 今回の予稿：条件付き定義にする

**現在の原稿（誤り）：** $\boldsymbol{\theta} = \{\mathbf{F}, \boldsymbol{\Sigma}, w_0^Y, w^Y\}$

**推奨する修正（案1：明示的条件付き）：**
> $\boldsymbol{\theta} = \{\mathbf{F},\ w_0^Y,\ w^Y\}$（$\mathrm{ExpFam}_X$ がガウス分布のとき属性分散 $\boldsymbol{\Sigma}$ も含む）

**推奨する修正（案2：脚注で補足）：**
本文では $\boldsymbol{\theta} = \{\mathbf{F}, w_0^Y, w^Y\}$ と書き，脚注に「Gaussian-X の場合は対角分散行列 Σ も含む」と記す。

**根拠：**  
- `model_dual_expfam.py` L.274-285: Bernoulli/Poisson-X の `calc_sigma` は `np.eye(self.d)` を返すのみで推定しない
- `calc_bic_dual` L.400: `sigma_x_p = d if family_x == "gaussian" else 0` → BIC でも非Gaussian時はカウントしない
- 常に θ に Σ を含めると，Bernoulli-X のモデルで「推定していないパラメータがある」ように読まれる

**V_X における Σ の記述は変更不要：**  
```
V_X(m_i) = Σ^{-1}                    (Gaussian-X)
         = diag(A_X''(F m_i))         (Bernoulli/Poisson-X)
```
この形は実装と一致しており，Σ が "Gaussian-X の場合のみ使用" という文脈で正確に示されている。

### 修論・将来版での整理

表形式でパラメータと推定法を整理した表（F，Σ，w0，w，σ_y，σ_z^2 の一覧）を入れると，先生や査読者への説明が容易になる。今回は字数の関係上本文の文章記述で代替。

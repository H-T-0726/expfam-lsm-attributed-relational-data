## 候補1：完全データ尤度

```
p(X,Y,Z|θ) = p(Z|θ) ∏_{i=1}^n ∏_{l=1}^d p_X(x_il|z_i,θ_X) ∏_{i<j} p_Y(y_ij|z_i,z_j,θ_Y)
```

### 判定：要修正（3か所）

**1. p(Z|θ) → p(Z) とすべき**

実装では `var_z = 1.0` は固定値であり推定しない（`initialize_params` で `var_z = 1.0` と設定、M-step で更新されない）。
よって z_i の事前分布 N(0, σ_z^2 I) の σ_z^2 = 1 は「超パラメータ」であり θ には含まれない。

修正後：p(Z) = ∏_{i=1}^n N(z_i; 0, I_k)（σ_z^2 = 1 固定）

**2. ∏_{i<j} の範囲は正しい**

Y は対称行列（y_{ij} = y_{ji}）を仮定しており、二重計上を避けるために i<j の積のみ使うのは正しい。
先行研究[1]の再現コード（reproduction/src/model.py）も同様の対称性を仮定している。

**3. θ の定義が実装と不一致（重要）**

原稿 eq(4) は η_{ij}^Y = z_i^T W_Y z_j（k×k 行列 W_Y を使用）と書いているが、
実装は η_{ij}^Y = w_0 + w z_i^T z_j（スカラー w_0, w を使用）である。

実装コード（model_dual_expfam.py, _calc_gradient, line ~153）:
```python
eta_y = w0 + w * (Z @ z_i)   # w0 = scalar, w = scalar
```

よって θ_Y = {w_0, w}（スカラー2個）が実装と整合する記法である。
また先行研究[1]（reproduction/src/model.py）も同じスカラー w_0, w を使用している。

X 側についても：原稿 eq(5) は η_{il}^X = w_{0l} + z_i^T w_l（バイアス w_{0l} あり）と書いているが、
実装は eta_x_i = F @ z_i（バイアスなし）であり θ_X = {F, Σ}（d×k 行列 F, 対角行列 Σ）。
先行研究[1]も同様に「Note: μ_x is omitted (assumed 0)」としている（model.py L.229）。

**修正後のθ：**
θ = {F, Σ, w_0, w} かつ σ_z^2 = 1 は固定

（ガウス Y の場合は σ_y も含む）

**先行研究[1]との対応：**
先行研究[1]は Bernoulli-Y + Gaussian-X に対して同一のスカラー (w_0, w) を使用。
提案手法はこれを指数型分布族に拡張したものであり、パラメータ構造は同一。

## 候補2：EMのQ関数

```
Q(θ|θ_old) = E_{p(Z|X,Y,θ_old)}[log p(X,Y,Z|θ)]
```

**先行研究[1]との対応：** NOLTA 2024 論文の中心的な目的関数に対応。

## 候補3：モンテカルロ近似

```
Q̂(θ|θ_old) = (1/R) Σ_{r=1}^R log p(X,Y,Z^(r)|θ),
Z^(r) ~ p(Z|X,Y,θ_old)
```

### 判定：要修正（2か所）

**1. R → L に変更必須**

実装・先行研究[1]ともに L を使用（DualExpFamLSM.__init__ 引数: `L: int = 10`）。
論文内で記号統一が必要。

**2. サンプリング手順の注記が必要**

Z^(l) は p(Z|X,Y,θ_old) から直接サンプリングするのではなく、ラプラス近似による近似分布から得る：

正確な手順：
1. 各 i について、ニュートン法でモード m_i = argmax_{z_i} log p(z_i|X,Y,Z_{-i},θ_old) を求める
2. 精度行列 A_i = -∇²_{z_i} log p(z_i|...) を計算
3. q_i(z_i) = N(m_i, A_i^{-1}) からサンプリング: z_i^(l) ~ q_i(z_i)
4. Z^(l) = [z_1^(l), ..., z_n^(l)]^T とする

これが「モンテカルロ EM + ラプラス近似（MCEM + Laplace）」の実体。

実装コード（utils_expfam.py, run_em_dual）:
```python
for l in range(L):
    Z_new = model.calc_eta_newton(X, Y, rng=rng, ...)  # ニュートン法 + サンプリング
    Z_samples[:, :, l] = Z_new
```

**先行研究[1]との対応：** 先行研究[1]の Algorithm 2（Lines 4-10）に対応。同一の手順を指数型分布族に拡張。

## 候補4：事後分布近似

```
q_i(z_i) = N(m_i, H_i^{-1})

H_i = -∇²_{z_i} log p(z_i|X,Y,Z_{-i},θ_old)|_{z_i=m_i}
```

### 判定：ほぼ正しい。ただし H_i の具体形を補記すべき

**定義式は正しい：**
- m_i = argmax_{z_i} log p(z_i|X,Y,Z_{-i},θ_old)（ニュートン法で探索）
- H_i = 精度行列（precision matrix）= 負のヘッセ行列

実装では H_i = A_i（_calc_precision_matrix の戻り値）として計算する。

**A_i の具体形（実装確認済み）：**

Gaussian X の場合：
A_i = (1/σ_z^2) I_k + F^T Σ^{-1} F + (w^2/2) Σ_{j≠i} A_Y''(η_{ij}^Y) z_j z_j^T

非Gaussian X（ベルヌーイ・ポアソン）の場合：
A_i = (1/σ_z^2) I_k + F^T diag(A_X''(F m_i)) F + (w^2/2) Σ_{j≠i} A_Y''(η_{ij}^Y) z_j z_j^T

ここで A_X''(·)、A_Y''(·) は各分布族の分散関数：
- ベルヌーイ: A''(η) = σ(η)(1-σ(η))（ロジスティック分散）
- ポアソン: A''(η) = exp(η)
- ガウス: A''(η) = 1（分散は別途 σ^2 として扱う）

**第1項（Z事前分布）、第2項（X尤度）、第3項（Y尤度）の3項構造が本研究の一般化の核心。**
先行研究[1]は X = Gaussian のみのため第2項が F^T Σ^{-1} F のみ。本研究では非Gaussian X にも対応。

**注意点：**
「Z_{-i} が固定」の意味について: 実装では z_i を逐次的に更新（i=0,1,...,n-1 の順）するため、
「現在の Z」の中には既に更新された z_0,...,z_{i-1} が含まれる。
厳密な z_{-i} 固定ではないが先行研究[1]と同じ慣行であり問題なし。

## 候補5：Mステップ

```
θ_new = argmax_θ Q̂(θ|θ_old)
```

### 判定：要修正（注釈が必要）

式自体は正しいが、各パラメータの更新法が異なるため一般形のみ書くと読者に「どうやって最大化するか」が伝わらない。

**実装での更新法（utils_expfam.py, run_em_dual）：**

| パラメータ | 条件 | 更新法 |
|-----------|------|--------|
| F（属性荷重行列, d×k） | family_x = 'gaussian' | 解析解（線形回帰）|
| F | family_x = 'bernoulli' or 'poisson' | Adam 勾配上昇（max_iter=50）|
| Σ（属性分散, 対角行列） | family_x = 'gaussian' | 解析解（残差分散）|
| Σ | 非Gaussian | 恒等行列（更新しない） |
| w_0, w（関係パラメータ, スカラー） | 常に | Adam 勾配上昇（max_iter=50）|
| σ_y（Y 側分散） | family_y = 'gaussian' のみ | 解析解（残差分散）|
| σ_z^2 | 全て | **1.0 固定（推定しない）** |

Gaussian X の F 解析解（calc_F, model.py L.526）:
F = (Σ_{l=1}^L X^T Z^(l)) (Σ_{l=1}^L Z^(l)^T Z^(l))^{-1}

非Gaussian X の F 勾配（_calc_F_adam, model_dual_expfam.py L.249）:
∇_F Q_X = (1/L) Σ_{l=1}^L (X - A_X'(Z_l F^T))^T Z_l

## 候補6：指数型分布族の対数尤度

```
log p_X(x_il|z_i) = log h_X(x_il) + η_il^X T_X(x_il) - A_X(η_il^X)
log p_Y(y_ij|z_i,z_j) = log h_Y(y_ij) + η_ij^Y T_Y(y_ij) - A_Y(η_ij^Y)
```

### 判定：ほぼ正しい。ガウス分布の扱いに注意

式(3)の展開として正しい。ただし以下に注意：

**ガウス分布（1パラメータ版）の取り扱い：**
ガウス分布は通常 2 パラメータ（μ, σ^2）だが、自然パラメータ形式では平均 μ を η、
分散 σ^2 を「分散パラメータ（dispersion parameter）φ」として分離して扱う：

p(x|η, φ) = h(x, φ) exp{η T(x) - A(η)} / φ

実装では:
- Gaussian X: η_{il}^X = (F z_i)_l（平均パラメータ）、分散 σ_l^2 は別途 Σ として推定
- Gaussian Y: η_{ij}^Y = w_0 + w z_i^T z_j（平均パラメータ）、分散 σ_y^2 は別途推定

2ページ予稿では「1パラメータ指数型分布族」と明記するか、または「ガウス分布では分散は既知または別途推定」と注記すれば問題なし。

**各分布族の十分統計量 T(x)と A(η)：**

| 分布族 | T(x) | A(η) | A'(η)（平均関数） | A''(η)（分散関数） |
|-------|------|------|-------------------|-------------------|
| ベルヌーイ | x | log(1+e^η) | σ(η) | σ(η)(1-σ(η)) |
| ポアソン | x | e^η | e^η | e^η |
| ガウス(φ=1) | x | η²/2 | η | 1 |

T(x) = x がすべての場合で共通（1パラメータ自然指数型分布族の特性）。
実装コードも `residual_x = x_i - mu_x_i` つまり `T_X(x) = x` を使用。

現状の原稿 eq(4):  η_{ij}^Y = z_i^T W_Y z_j（行列 W_Y）

4．実験

4.1 実験条件

提案手法の有効性を示すため，人工データを用いた実験を行った．オブジェクト数 $n = 150$，属性次元数 $d = 15$，真の潜在次元数 $k^* = 3$ とし，指定した分布族に従う人工データを生成した．実験では，関係データと属性データの分布族の組み合わせとして，シナリオ A（$\mathrm{ExpFam}_X$ = ポアソン，$\mathrm{ExpFam}_Y$ = ベルヌーイ），シナリオ B（$\mathrm{ExpFam}_X$ = ガウス，$\mathrm{ExpFam}_Y$ = ポアソン），シナリオ C（$\mathrm{ExpFam}_X$ = ベルヌーイ，$\mathrm{ExpFam}_Y$ = ガウス）の 3 種類を設定し，各条件で 10 試行の平均を用いた．評価指標は，プロクルステス回転を適用した潜在変数行列の推定誤差 $\mathrm{RMSE}(\mathbf{Z})$ とした．

4.2 実験結果と考察

情報量基準による次元選択では，3 種類のシナリオすべてにおいて BIC が真の潜在次元数 $k^* = 3$ を正しく選択した．

分布族の誤指定が推定精度に与える影響を評価した結果を図1に示す．各シナリオについて，正しい分布族指定時の $\mathrm{RMSE}(\mathbf{Z})$ を基準（1.0 倍）として誤指定時の悪化倍率を算出した．いずれのシナリオでも誤指定により推定精度が大きく低下し，最大悪化倍率はシナリオ A で 3.4 倍，B で 7.3 倍，C で 41.5 倍であった．また，従来手法に相当するガウス-$X$・ベルヌーイ-$Y$ 固定の指定においても，シナリオ B で 6.5 倍，シナリオ C で 23.6 倍の推定精度の低下が確認された．なお，シナリオ C では $\mathrm{ExpFam}_Y$ = ガウス分布が推定に支配的に寄与するため，$X$ の誤指定の影響が相対的に小さくなる条件も存在した．以上より，観測データの分布特性に応じた分布族の選択が潜在構造推定の精度に大きく影響することが示された．

先行研究[1]と同一の分布族条件（ガウス-$X$，ベルヌーイ-$Y$）では，提案手法は先行研究[1]と実質的に同等の推定精度を維持した（$\mathrm{RMSE}(\mathbf{Z})$ の差 < 0.001）．

3.2 パラメータ推定

提案モデルのパラメータ $\boldsymbol{\theta} = \{\mathbf{F},\ w_0^Y,\ w^Y\}$ は，モンテカルロ EM（MCEM）とラプラス近似[1]を指数型分布族に拡張した手法で推定する．潜在変数の事前分散は $\sigma_z^2 = 1$ と固定する[1]．

E ステップでは，各 $\mathbf{z}_i$ の事後分布をガウス分布 $q_i(\mathbf{z}_i) = \mathcal{N}(\mathbf{m}_i, \mathbf{A}_i^{-1})$ でラプラス近似し，$L$ 個のモンテカルロサンプルを生成する．精度行列 $\mathbf{A}_i$ は

$$\mathbf{A}_i = \mathbf{I}_k + \mathbf{F}^\top \mathbf{V}_X(\mathbf{m}_i)\,\mathbf{F} + \frac{(w^Y)^2}{2} \sum_{j \neq i} A_Y''(\eta_{ij}^Y)\, \mathbf{z}_j \mathbf{z}_j^\top \tag{5}$$

と表される．ここで $\mathbf{V}_X(\mathbf{m}_i)$ は $\mathrm{ExpFam}_X$ に応じた行列であり，
ガウス分布のとき $\boldsymbol{\Sigma}^{-1}$（先行研究[1]と同一形），
ベルヌーイ・ポアソンのとき $\mathrm{diag}(A_X''(\mathbf{F}\mathbf{m}_i))$ となる（$A''(\cdot)$：各分布族の分散関数）．式(5)の第2項が本研究による一般化の核心であり，先行研究[1]の Gaussian-X 専用の精度行列を任意の $\mathrm{ExpFam}_X$ に拡張する．

M ステップでは，$L$ 個のサンプルで近似した Q 関数を $\boldsymbol{\theta}$ について最大化する．$\mathrm{ExpFam}_X$ = ガウス分布のとき F は先行研究[1]と同一の解析解，それ以外では Adam 勾配上昇法で更新する．$w_0^Y, w^Y$ は常に Adam で更新する[1]．潜在次元数 $k$ の選択には BIC を用いる．

---

3.2 パラメータ推定

提案モデルのパラメータを，モンテカルロ EM（MCEM）とラプラス近似[1]を指数型分布族に拡張した手法で推定する．EM アルゴリズムは以下の Q 関数を最大化する：

$$Q(\boldsymbol{\theta} \mid \boldsymbol{\theta}^{\mathrm{old}}) \approx \frac{1}{L}\sum_{l=1}^L \log p(\mathbf{X}, \mathbf{Y}, \mathbf{Z}^{(l)} \mid \boldsymbol{\theta}) \tag{5}$$

ここで $\mathbf{Z}^{(l)}$ はラプラス近似による事後分布 $q_i(\mathbf{z}_i) = \mathcal{N}(\mathbf{m}_i, \mathbf{A}_i^{-1})$ からのモンテカルロサンプルである．精度行列 $\mathbf{A}_i$ は

$$\mathbf{A}_i = \mathbf{I}_k + \mathbf{F}^\top \mathbf{V}_X(\mathbf{m}_i)\,\mathbf{F} + \frac{(w^Y)^2}{2}\sum_{j \neq i} A_Y''(\eta_{ij}^Y)\,\mathbf{z}_j \mathbf{z}_j^\top \tag{6}$$

と表され，$\mathbf{V}_X$ は $\mathrm{ExpFam}_X$ の分布族に依存する（ガウス: $\boldsymbol{\Sigma}^{-1}$，
ベルヌーイ・ポアソン: $\mathrm{diag}(A_X''(\mathbf{F}\mathbf{m}_i))$）．
式(6)の第2項が X 側の一般化であり，先行研究[1]の精度行列を任意の $\mathrm{ExpFam}_X$ に拡張する．M ステップでは，F を解析解（$\mathrm{ExpFam}_X$=ガウス）または Adam（その他）で，$w_0^Y, w^Y$ を Adam で更新する[1]．次元数 $k$ の選択には BIC を用いる．

3.2 パラメータ推定

提案モデルのパラメータは，先行研究[1]のモンテカルロ EM（MCEM）＋ラプラス近似をベースに，E ステップの精度行列を指数型分布族に拡張した手法で推定する．
各オブジェクトの潜在変数 $\mathbf{z}_i$ の事後分布をガウス分布 $\mathcal{N}(\mathbf{m}_i, \mathbf{A}_i^{-1})$ で近似し，精度行列を

$$\mathbf{A}_i = \mathbf{I}_k + \mathbf{F}^\top \mathbf{V}_X \mathbf{F} + \frac{(w^Y)^2}{2}\sum_{j \neq i} A_Y''(\eta_{ij}^Y)\,\mathbf{z}_j \mathbf{z}_j^\top \tag{5}$$

と定義する（$\mathbf{V}_X$ は $\mathrm{ExpFam}_X$ の分散関数に基づく重み行列）．
この近似分布からモンテカルロサンプルを生成し，Q 関数を最大化してパラメータを更新する．
次元数 $k$ の選択には BIC を用いる．

## 10. 実験の目的

この実験の目的は、提案手法が単に動くことを確認するだけではなく、**データ型に応じて分布族を選ぶことの重要性を確認すること**である。

以下の 5 点を確認することを目的とする。

| # | 実験の目的 |
|---|-----------|
| 1 | 複数の分布族の組み合わせで、提案手法が潜在構造 $Z$ を推定できるか |
| 2 | BIC が真の潜在次元 $k^* = 3$ を正しく選択できるか |
| 3 | サンプル数 $n$ が増えると推定精度（RMSE(Z)）が改善するか |
| 4 | 分布族を誤指定したとき、RMSE(Z) がどの程度悪化するか |
| 5 | 先行研究と同じ条件（Gauss-X, Bern-Y）で、提案手法が既存手法と実質同等に動くか |

実験で重視するのは「**正しい分布族を指定することがどれだけ重要か**」である。  
先行研究は分布族を固定しているため、データ型が変わった場合に誤指定が生じる。提案手法により、データ型に応じて分布族を選択できる枠組みを用いることで、分布族固定による誤指定の影響を軽減できる可能性を検証する。

> **根拠ファイル：** `conference_submission_final_draft.md` §4.1、`docs_for_notebooklm/NOTEBOOKLM_RESEARCH_BRIEF.md` §10

## 11. 実験設定

### 11.1 共通設定

| 項目 | 値 | 補足 |
|-----|---:|------|
| オブジェクト数 $n$ | 150 | n-sweep 実験では 50〜300 で変化 |
| 属性次元数 $d$ | 15 | — |
| 真の潜在次元数 $k^*$ | 3 | 推定では $k \in \{1,2,3,4,5,6\}$ を探索 |
| 試行数 | 10 | 各条件で 10 回実施し平均を評価 |
| MC サンプル数 $L$ | 5 | E-step の Laplace 近似サンプル数 |
| EM 反復数 | 8 | E-step → M-step を 8 回繰り返す |

### 11.2 Scenario A / B / C

| Scenario | 真の $\mathrm{ExpFam}_X$ | 真の $\mathrm{ExpFam}_Y$ | 確認すること |
|---------|:--------------------:|:--------------------:|------------|
| **A（P-B）** | Poisson | Bernoulli | 先行研究と同じ Y=Bernoulli を保ちながら、X 側をカウント属性（Poisson）に拡張できるか |
| **B（G-P）** | Gaussian | Poisson | X は先行研究と同じ Gaussian のまま、Y 側をカウント関係（Poisson）に拡張できるか |
| **C（B-G）** | Bernoulli | Gaussian | X を二値属性（Bernoulli）、Y を連続値関係（Gaussian）として、先行研究とは大きく異なる組み合わせでも動くか |

### 11.3 評価指標

| 指標 | 内容 |
|-----|------|
| RMSE(Z) | 潜在変数 $Z$ の推定精度。**主指標** |
| RMSE(X) | 属性データの再構成精度 |
| RMSE(Y) | 関係データの再構成精度 |
| BIC | $-2\hat{Q}_{\mathrm{strict}} + \mathrm{num\_params} \cdot \ln n$。潜在次元 $k$ の選択に使用 |

**Procrustes 回転について：**  
潜在変数 $Z$ は回転しても同じ構造を表すため、推定した $Z_\mathrm{est}$ と真の $Z_\mathrm{true}$ を直接比較せず、Procrustes 回転で向きを合わせてから RMSE を計算する（`utils_expfam.py::procrustes_rotation`）。全実験で適用済み。

> **根拠ファイル：** `expfam/README.md` §2、`expfam/src/exp_scenario_lib.py`（L.40-45）

## 12. 実験結果

### 12.1 k-sweep / BIC の結果

10 試行平均 BIC で、**全シナリオで $k = k^* = 3$ が選択された**。  
また、$k = 3$ のときに RMSE(Z) が最小値をとり、$k^*$ を正しく識別できていることが確認できた。

| Scenario | 真の分布 | RMSE(Z) at $k$=3 | 10試行平均 BIC の選択 | 注意点 |
|---------|---------|----------------:|---------------------|------|
| A (P-B) | Pois-X / Bern-Y | **0.278** | $k = 3$ ✓ | BIC 差（$k$=3 vs $k$=4）= 487、選択は安定 |
| B (G-P) | Gauss-X / Pois-Y | **0.182** | $k = 3$ ✓ | BIC 差 = **180**。他シナリオより小さく、**安定性に注意** |
| C (B-G) | Bern-X / Gauss-Y | **0.028** | $k = 3$ ✓ | BIC 差 = 484（負値は Gaussian-Y の正規化定数由来、正常） |

### 12.4 図 1(b) の 23.6× と 41.5× の注意

図 1(b) は代表的な **3 種の条件**（X-side 誤指定の最大値・Y-side 誤指定の最大値・先行研究固定条件）を表示しており、全誤指定条件を網羅したものではない。

| 値 | 条件 | 意味 | 図に表示 |
|---:|------|------|:------:|
| **23.6×** | Scen. C・Fixed Gauss-X/Bern-Y（X=Gaussian, Y=Bernoulli） | 先行研究固定条件を Scen. C に適用したときの悪化倍率 | ✓ 灰色バー（図の視覚上の最大値） |
| **41.5×** | Scen. C・X=Gaussian, Y=Poisson（**X・Y 両方誤指定**） | 全誤指定条件中の最大悪化倍率（CSV 計算値） | **✗ 独立したバーなし** |

**ポイント：**
- 図を見た人が「最大が 23.6×」と誤解する可能性がある
- 原稿本文（L.83）の「最大 41.5 倍」は図には直接対応するバーがない
- 41.5× は「X・Y 両方を最悪の組み合わせで誤指定した場合」の値であり、図に示した条件とは種別が異なる

### 12.5 Control 条件の結果

先行研究と同じ条件（$\mathrm{ExpFam}_X$ = Gaussian、$\mathrm{ExpFam}_Y$ = Bernoulli）で提案手法と先行研究を比較した。

| モデル | RMSE(Z) 平均 | RMSE(Z) 標準偏差 |
|-------|:-----------:|:--------------:|
| 先行研究（baseline） | 0.179 | ± 0.014 |
| 提案手法（Dual-ExpFam LSM） | 0.180 | ± 0.016 |
| 差（絶対値） | **0.0006** | — |

RMSE(Z) の差は 0.0006 であり、原稿では「0.001 未満」と表現している。  
先行研究と同じ条件では、**既存研究と同等の性能を保ちながら、一般化した枠組みとして動作することを確認した**結果である。  
（「完全に同一」「差がゼロ」ではなく、「実質同等」という表現が適切。）

### 12.6 Scenario C の慎重な解釈

Scenario C（真の X=Bernoulli, Y=Gaussian）では、以下の結果が得られた。

| 条件 | RMSE(Z) | Proposed に対する倍率 | 解釈 |
|-----|:-------:|:------------------:|------|
| Proposed（Bern-X, Gauss-Y） | **0.0287** | 1.00× | 基準条件 |
| Y-only（X 情報を除去） | 0.0286 | ≈ 1.00× | X 情報なしでも Z 推定はほぼ同等 |
| X=Poisson（X 側のみ誤指定） | 0.0284 | ≈ 0.99× | X 誤指定でも大きな悪化なし |
| X=Gaussian（X 側のみ誤指定） | 0.1055 | 3.67× | Gaussian-X では大きく悪化 |
| X-only（Y 情報を除去） | 1.1024 | **38.4×** | Y なしでは Z 推定が壊滅的に悪化 |
| Y=Bernoulli（Y 側のみ誤指定） | 0.4523 | 15.75× | Y の誤指定で大きく悪化 |

**観察：**  
Y-only 条件（X を除去）でも Proposed とほぼ同等の RMSE(Z) が得られた一方、X-only 条件では大幅に悪化した。

**解釈（断定しない）：**  
Y=Gaussian は内積 $z_i^\top z_j$ を連続値で直接観測するため（$n(n-1)/2 = 11,175$ 個）、Bernoulli X（$n \cdot d = 2,250$ 個、二値）に比べて $Z$ の復元に強く寄与した**可能性がある**。  
ただし、E-step 中の $\|$Term2$\|$（X 側）と $\|$Term3$\|$（Y 側）の定量比較は未実施であり、**「Y が支配的」とは断定しない**。追加検証が必要である。

> **根拠ファイル：** `expfam/results/exp_scenario_C_exp4_mismatch.csv`、`docs_for_notebooklm/02_experiment_result_verification.md` §12

## 13. 実験から言えること・慎重に扱うべきこと

### 13.1 言えること

以下は CSV 照合済みの事実として確認できた内容である。

1. **提案手法が複数の分布族の組み合わせで動作した：** Scenario A/B/C（各シナリオで正解族を指定した条件）で RMSE(Z) が小さく、潜在構造の推定が機能している
2. **10 試行平均 BIC で全シナリオが $k^* = 3$ を選択した：** BIC による次元選択が機能している（ただし Scen. B の差は 180 と小さい）
3. **$n$ が増えると RMSE(Z) が改善する傾向が確認できた：** 3 シナリオで削減率 31〜62%
4. **分布族の誤指定により RMSE(Z) が大きく悪化する：** 全条件中の最大倍率は Scen. C で 41.5 倍（CSV 計算値）
5. **先行研究と同じ条件では、提案手法は実質同等に動作した：** RMSE(Z) 差 0.0006（0.001 未満）

「有効性を完全に証明した」とは言えない。実データへの適用、アルゴリズムの精査、追加実験が今後必要である。

### 13.2 慎重に扱うべきこと

| 主張 | 注意事項 |
|-----|---------|
| 「X 情報が Z 推定に有効に活用されている」 | Scen. C で Y-only ≈ Proposed のため、X の寄与が計測できていない。$\|$Term2$\|$ vs $\|$Term3$\|$ の定量比較が必要 |
| 「Y=Gaussian が Z 推定を支配している」 | 可能性はあるが、Term2/Term3 定量比較は未実施。**断定しない** |
| 「BIC が常に $k=3$ を選ぶ」 | Scen. B の BIC 差は 180 と相対的に小さく、外れ試行では $k=4$ が選ばれる可能性がある。「10 試行平均 BIC では選択された」と表現する |
| 「41.5 倍を図から確認できる」 | 図 1(b) の最大バーは 23.6×（灰色）。41.5× は図に独立したバーとして表示されていない |
| 「先行研究との差がゼロ」 | 差は 0.0006。「実質同等」または「0.001 未満」が適切な表現 |
| 「Python 実装の Newton 方向が原稿式と一致している」 | E-step の Y 側 Term3 に余分な 0.5 が残存。アルゴリズム精査の課題として認識している |

1．研究背景・目的

近年，ソーシャルネットワーク，購買履歴，推薦システムなど，多様な分野において，対象間の関係を表す関係データ[3]と，各対象が持つ属性データが同時に観測されることが多い．このようなデータの統合的な分析を行うための手法として，関係データと属性データの双方から共通の低次元潜在変数を推定する属性情報付き潜在構造モデルが提案されている[1]．しかし，従来の属性情報付き潜在構造モデルでは，関係データの分布にベルヌーイ分布，属性データの分布にガウス分布を仮定するなど，観測データの分布族が固定されている場合が多い．そのため，訪問回数や購買量のようなカウントデータ，あるいは異なる種類のデータが混在する実データへ適用する際には，データ型ごとにモデルを再定式化する必要があるという問題がある．

そこで本研究では，関係データと属性データの双方を指数型分布族に基づいて記述することで，多様なデータ型を統一的に扱える属性情報付き潜在構造モデルを提案する．人工データを用いた実験により，提案手法の有効性を評価する．


2．準備

2.1 関係データと属性データ

$n$ 個のオブジェクトからなる集合を考える．関係データは，オブジェクト $i$ と $j$ の間の関係を表す行列 $\mathbf{Y} = (y_{ij})$ で与えられる（$1 \le i \le n$，$1 \le j \le n$）．$y_{ij}$ の値域はデータの種類に応じて定まり，例えば $y_{ij} \in \{0, 1\}$（二値関係）や $y_{ij} \in \{0, 1, 2, \ldots\}$（カウント関係）を含む．また，各オブジェクト $i$ が $d$ 次元の属性を持つとし，属性データ行列を $\mathbf{X} = (x_{il})$（$1 \le l \le d$）と表す．

2.2 属性情報付き潜在構造モデル

属性情報付き潜在構造モデル[1]は，$\mathbf{Y}$ と $\mathbf{X}$ の双方から，共通の $k$ 次元潜在変数行列 $\mathbf{Z} \in \mathbb{R}^{n \times k}$ を推定するモデルである．いま，オブジェクト $i$ に対応する潜在ベクトルを $\mathbf{z}_i \in \mathbb{R}^k$ とする．先行研究[1]の設定を参考に，本研究では関係データと属性データの生成過程を以下の形式で考える．

$$y_{ij} \mid \mathbf{z}_i, \mathbf{z}_j \sim \mathrm{Bernoulli}\!\left(\sigma\!\left(w_0 + w\, \mathbf{z}_i^\top \mathbf{z}_j\right)\right) \tag{1}$$

$$x_{il} \mid \mathbf{z}_i \sim \mathcal{N}\!\left(\mathbf{f}_l^\top \mathbf{z}_i,\ \sigma_l^2\right) \tag{2}$$

ただし，$\sigma(\cdot)$ はシグモイド関数，$w_0, w \in \mathbb{R}$ は関係データのスカラーパラメータ，$\mathbf{F} \in \mathbb{R}^{d \times k}$ は属性荷重行列（$\mathbf{f}_l$ はその第 $l$ 行ベクトル），$\sigma_l^2$ は属性 $l$ の分散である．先行研究[1]と同様，属性データのバイアスは $0$ と仮定する．

2.3 指数型分布族

指数型分布族に属する確率分布は，次の形で表される[2]．

$$p(x \mid \eta) = h(x) \exp\!\left\{\eta\, T(x) - A(\eta)\right\} \tag{3}$$

ここで，$\eta$ は自然パラメータ，$T(x)$ は十分統計量，$A(\eta)$ は対数分配関数，$h(x)$ はベース測度である．ベルヌーイ分布，ポアソン分布，ガウス分布はいずれも式(3)の特例として表現できる[2]．本稿では各観測値を1変量の指数型分布族として扱い，属性情報については各属性次元 $l$ ごとに独立に適用する．


3．提案手法

従来の属性情報付き潜在構造モデル[1]では，式(1),(2)に示すように関係データにベルヌーイ分布，属性データにガウス分布が固定的に仮定されているため，これら以外の分布族を持つデータへ適用する際にはモデルの再定式化が必要となるという問題がある．そのため本研究では，関係データと属性データの双方に指数型分布族を導入し，分布族の種類を分析者が柔軟に指定できるモデルを定式化する．

3.1 関係データの指数型分布族への拡張

提案モデルでは，関係データ $y_{ij}$（$i < j$）を，自然パラメータ $\eta_{ij}^Y$ を持つ指数型分布族 $\mathrm{ExpFam}_Y$ に従う確率変数として扱う．

$$y_{ij} \mid \mathbf{z}_i, \mathbf{z}_j \sim \mathrm{ExpFam}_Y\!\left(\eta_{ij}^Y\right), \quad \eta_{ij}^Y = w_0^Y + w^Y \mathbf{z}_i^\top \mathbf{z}_j \quad (i < j) \tag{4}$$

$\mathrm{ExpFam}_Y$ には，ベルヌーイ分布（二値関係），ポアソン分布（カウント関係），ガウス分布（連続値関係）などを指定できる．$\mathrm{ExpFam}_Y$ にベルヌーイ分布を指定した場合，式(4)は式(1)の特殊ケースとして表せる．

3.2 属性データの指数型分布族への拡張

属性データについても同様に，各属性次元 $l$ の観測値 $x_{il}$ を指数型分布族 $\mathrm{ExpFam}_X$ で記述する．

$$x_{il} \mid \mathbf{z}_i \sim \mathrm{ExpFam}_X\!\left(\eta_{il}^X\right), \quad \eta_{il}^X = \mathbf{f}_l^\top \mathbf{z}_i \tag{5}$$

先行研究[1]と同様，属性データのバイアスは $0$ と仮定する（$\mathbf{f}_l^\top \mathbf{z}_i = \eta_{il}^X$）．$\mathrm{ExpFam}_X$ にガウス分布を指定すると，式(5)は式(2)の特殊ケースとして表せる．$\mathrm{ExpFam}_Y$ と $\mathrm{ExpFam}_X$ はそれぞれ独立に指定できるため，関係データと属性データが異なる分布族を持つ場合にも，一つの枠組みで統一的に扱える．

3.3 パラメータ推定

提案モデルのパラメータ $\boldsymbol{\theta} = \{\mathbf{F},\ w_0^Y,\ w^Y\}$ は，モンテカルロ EM（MCEM）とラプラス近似[1]を指数型分布族に拡張した手法で推定する（$\mathrm{ExpFam}_X$ がガウス分布のとき，対角分散行列 $\boldsymbol{\Sigma}$ も含む）．潜在変数の事前分散は識別性のため $\sigma_z^2 = 1$ と固定する[1]．

E ステップでは，各 $\mathbf{z}_i$ の事後分布をガウス分布 $q_i(\mathbf{z}_i) = \mathcal{N}(\mathbf{m}_i, \mathbf{A}_i^{-1})$ でラプラス近似し，$L$ 個のモンテカルロサンプルを生成する．精度行列 $\mathbf{A}_i$ は

$$\mathbf{A}_i = \mathbf{I}_k + \mathbf{F}^\top \mathbf{V}_X(\mathbf{m}_i)\,\mathbf{F} + (w^Y)^2 \sum_{j \neq i} A_Y''(\eta_{ij}^Y)\, \mathbf{z}_j \mathbf{z}_j^\top \tag{6}$$

と表される．ここで $\mathbf{V}_X(\mathbf{m}_i)$ は $\mathrm{ExpFam}_X$ に応じた行列であり，ガウス分布のとき $\boldsymbol{\Sigma}^{-1}$（先行研究[1]と同一形），ベルヌーイ・ポアソンのとき $\mathrm{diag}(A_X''(\mathbf{F}\mathbf{m}_i))$ となる（$A''(\cdot)$：分布族の分散関数）．式(6)の第2項が本研究による一般化の核心であり，先行研究[1]の精度行列を任意の $\mathrm{ExpFam}_X$ に拡張する．

M ステップでは，$L$ 個のサンプルで近似した Q 関数を $\boldsymbol{\theta}$ について最大化する．$\mathrm{ExpFam}_X$ がガウス分布のとき $\mathbf{F}$ は先行研究[1]と同一の解析解，それ以外では Adam 勾配上昇法で更新する．$w_0^Y, w^Y$ は常に Adam で更新する[1]．潜在次元数 $k$ の選択には BIC を用いる．


4．実験

4.1 実験条件

提案手法の有効性を示すため，人工データを用いた実験を行った．オブジェクト数 $n = 150$，属性次元数 $d = 15$，真の潜在次元数 $k^* = 3$ とし，指定した分布族に従う人工データを生成した．関係データと属性データの分布族の組み合わせとして，シナリオ A（$\mathrm{ExpFam}_X$ = ポアソン，$\mathrm{ExpFam}_Y$ = ベルヌーイ），シナリオ B（$\mathrm{ExpFam}_X$ = ガウス，$\mathrm{ExpFam}_Y$ = ポアソン），シナリオ C（$\mathrm{ExpFam}_X$ = ベルヌーイ，$\mathrm{ExpFam}_Y$ = ガウス）の 3 種類を設定し，各条件で 10 試行の結果を平均した．情報量基準による次元選択では $k \in \{1, 2, 3, 4, 5, 6\}$ の範囲を探索した．また，$n$ を 50 から 300 の範囲で変化させた実験も行った．評価指標には潜在変数行列の推定精度 $\mathrm{RMSE}(\mathbf{Z})$ を主指標とし，データ再構成精度 $\mathrm{RMSE}(\mathbf{Y})$，$\mathrm{RMSE}(\mathbf{X})$ を補助指標として用いた．潜在変数行列の比較にはプロクルステス回転を適用した上で誤差を算出した．

4.2 実験結果と考察

情報量基準による次元選択の結果を表1に示す．3 種類のシナリオ全てにおいて $k = 3$ のとき $\mathrm{RMSE}(\mathbf{Z})$ が最小値をとり，BIC による次元選択も全シナリオで $k^* = 3$ を正しく与えた．

$n$ を変化させた場合の $\mathrm{RMSE}(\mathbf{Z})$ の変化を図1に示す．$n = 50$ から $n = 300$ にかけて全体として推定精度が改善し，削減率はシナリオ A で 48.8%（0.406 → 0.208），B で 31.0%（0.190 → 0.131），C で 62.0%（0.053 → 0.020）であった．なお，シナリオ B では $n = 50$ から $n = 100$ の間に一時的な変動が見られた．

分布族の誤指定が推定精度に与える影響を評価した結果を図2に示す．誤った分布族を指定した条件では正解条件と比較して $\mathrm{RMSE}(\mathbf{Z})$ が顕著に悪化し，最大悪化倍率はシナリオ A で 3.41 倍，B で 7.35 倍，C で 41.5 倍であった．なお，シナリオ C では $\mathrm{ExpFam}_Y$ = ガウス分布が推定に支配的な寄与をするため，$\mathrm{ExpFam}_X$ をポアソンに誤指定した場合でも正解条件と同程度の精度が得られる条件が存在した．

先行研究[1]が直接適用可能な条件（$\mathrm{ExpFam}_Y$ = ベルヌーイ，$\mathrm{ExpFam}_X$ = ガウス）での比較を表2に示す．両モデルの $\mathrm{RMSE}(\mathbf{Z})$ の差は 0.001 未満であり，提案手法は先行研究[1]と実質的に同等の推定精度を維持した．以上より，提案手法は先行研究[1]の条件では同等の推定性能を保ちつつ，多様な分布族の組み合わせへの適用が可能であることが示された．

5．まとめ

本研究では，関係データと属性データの双方を指数型分布族に基づいて記述することで，多様なデータ型を統一的に扱える属性情報付き潜在構造モデルを提案した．人工データを用いた実験において，情報量基準による潜在次元の適切な選択，サンプル数の増加に伴う推定精度の改善，および正確な分布族指定の重要性を確認した．また，先行研究[1]と同一の分布族条件では提案手法が同等の推定精度を示し，従来モデルの機能を損なうことなく指数型分布族への拡張が実現できることを示した．今後の課題として，実データへの適用，パラメータ推定アルゴリズムの詳細な整備，および分布族の誤指定が推定性能に与える影響のさらなる分析が挙げられる．

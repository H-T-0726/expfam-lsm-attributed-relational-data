## 3. 数理的・実装的な懸念点 (Roadblocks)

### (1) RMSE(X) が n・d によらずほぼ一定 (~0.31)
**原因**: X の再構成は `Z_est @ F_est.T` で行うが、X 自体が `Z_true @ F_true.T + noise(Sigma)` で生成されており、ノイズ項 Sigma (対角 0.1) が再構成誤差のフロアを形成する。n の増加は Z・F の推定精度を上げるが、観測ノイズそのものは除去できないため RMSE(X) は Sigma の対角値 (~sqrt(0.1)≈0.316) に収束する。論文には「RMSE(X) の下限はデータ生成ノイズに起因する」と明記できる。

### (2) Poisson RMSE(F) の d=15→20 での跳ね上がり (0.035→0.092)
**観測**: Exp2 の Poisson で d=20 以降、RMSE(F) が約2.6倍に増加。Z・Sigma・w0・w は引き続き改善しているため、F の Procrustes アライメントに局所解が生じている可能性が高い。num_iter=10 では d が大きい場合の F 収束が不十分なケースがある。**論文上の影響**: 他の指標は正常。Appendix に「Poisson モデルでは d が大きい場合に F の収束に追加イテレーションが有効」と注記可能。

### (3) RMSE(Y) の定義について
RMSE(Y) は「推定パラメータから計算した予測平均 μ_est」と「真のパラメータから計算した真の平均 μ_true」の差を比較。Poisson の RMSE(Y) が Bernoulli・Gaussian の約10倍大きく見えるのは、Poisson の平均 λ=exp(η) が 1~5 程度の値を取るため絶対誤差スケールが大きくなるためで、正規化すれば同等精度である。

### (4) Gaussian の圧倒的高精度
連続観測は情報量が最大のため、全指標で Bernoulli・Poisson の 1/5~1/10 の誤差。これは理論通りであり、論文中で「連続値観測への拡張により最高精度を達成」という主張を裏付ける。

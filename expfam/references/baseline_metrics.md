## ⚠️ 重要な前提の違い

| 項目 | Mikawa et al. (ベースライン) | Dual-ExpFam LSM (本研究) |
|------|--------------------------|------------------------|
| Y の分布 | Bernoulli **固定** | 任意指数型分布族（設定可能） |
| X の分布 | Gaussian **固定** | 任意指数型分布族（設定可能） |
| 推定対象 | Z, F, Sigma, w0, w | 上記 + family_x, family_y |
| BIC num_params | k*d - k*(k-1)//2 + d | k*d - k*(k-1)//2 + d + [family dependent] |

**Scenario A [Poisson×Bernoulli] はベースラインに最も近い**（Y=Bernoulli 共通）が、
X=Poisson（非Gaussian）の点で設定が異なる。
**直接的な数値比較が有効なのは**、Gaussian X + Bernoulli Y の場合のみ（本研究には含まない）。

## 検証者へのコメント

### 比較時の注意点

1. **直接比較はできない**: Dual-ExpFam は X の分布が変わるため、ベースラインとの RMSE 直接比較は公平でない。
   Dual-ExpFam の主張は「正しい family 指定が誤った指定より優れる」であり、
   ベースラインとの優劣ではない。

2. **Scenario A の RMSE(Z) がベースライン再現より悪い**:
   - Baseline (Y-only, Gaussian X) at n=150: RMSE(Z) = 0.2018
   - Scenario A (Dual Poisson X + Bernoulli Y) at n=150: RMSE(Z) = 0.2784
   - X=Poisson（非Gaussian）は X 側の E-step が近似的になるため悪化するのは理解できるが、
     **本当に X の情報が有効活用されているか検証が必要。**

3. **Scenario C の X mismatch が軽微な謎**:
   - X=Poisson (誤) が Proposed (Bern) とほぼ同等 (0.99×)
   - No X ablation も 1.00×
   - これは「Y=Gaussian が支配的で X 情報が無視されている」のか、
     それとも X 側の実装バグで X 情報が実際には使われていないのかを区別すべき。
